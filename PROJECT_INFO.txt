项目名称: SoterService 拦截器
作者: 王听
版本: 1.0
创建日期: 2024年

================================================================================

项目概述:
---------
本项目是一个Android Xposed框架模块，用于拦截com.chunqiunativecheck应用
对SoterService.apk的调用。该模块仅供学习研究使用，请勿用于违法行为。

================================================================================

项目结构:
---------
SoterServiceInterceptor/
├── app/
│   ├── build.gradle                    # App模块构建配置
│   └── src/main/
│       ├── AndroidManifest.xml         # Android清单配置
│       ├── assets/
│       │   └── xposed_init             # Xposed模块初始化文件
│       └── java/com/chunqiunativecheck/interceptor/
│           ├── SoterInterceptor.java       # 主拦截模块
│           └── SoterInterceptorSettings.java  # 设置页面
├── build.gradle                        # 根项目构建配置
├── settings.gradle                     # 项目设置
├── gradle.properties                   # Gradle属性配置
├── README.md                           # 详细使用说明
├── TECHNICAL.md                        # 技术文档
└── USAGE.txt                           # 快速使用指南

================================================================================

核心功能:
---------
1. SoterService API拦截
   - getSoterAppInfo()
   - isSupportSoter()
   - requestAvailabilityAndSign()

2. 系统服务监控
   - ServiceManager.getService()
   - Soter系统服务连接

3. Native层监控
   - nativeCheckInit
   - nativeCheckUpdate
   - nativeCheckVerify
   - nativeCheckRelease

4. 日志记录
   - 完整的调用栈追踪
   - 参数和返回值记录
   - Xposed日志系统集成

================================================================================

技术特性:
---------
- Xposed框架集成
- Java反射API使用
- 系统服务Hook
- Native方法监控
- 完整堆栈追踪
- 可配置拦截规则

================================================================================

编译要求:
---------
- Android SDK API 30
- Gradle 7.0+
- Android Studio 4.0+
- JDK 8+
- Xposed API 82

================================================================================

安装步骤:
---------
1. 使用Android Studio打开项目
2. 编译生成APK: ./gradlew assembleRelease
3. 安装APK到已Root设备
4. 在Xposed/LSPosed管理器中启用模块
5. 勾选目标应用 com.chunqiunativecheck
6. 重启设备或目标应用
7. 在Xposed日志中查看拦截记录

================================================================================

使用方法:
---------
1. 查看日志:
   - 打开LSPosed/Xposed管理器
   - 进入日志功能
   - 搜索 "SoterInterceptor"
   - 查看详细的调用记录

2. 配置选项:
   - 启用/禁用日志记录
   - 选择性拦截方法
   - 堆栈跟踪开关

================================================================================

注意事项:
---------
⚠️ 警告: 该模块仅用于学习研究，请勿用于违法行为，出现问题自负。

- 需要Root权限
- 需要Xposed框架支持
- 仅监控，不修改返回值
- 记录所有调用细节
- 可能无法拦截所有调用
